# 使い方・実行手順

## アプリの起動

```bash
# 開発サーバーを起動
npm run dev
```

起動後、ターミナルに表示された URL（通常 `http://localhost:5173`）をブラウザで開く。

## 使用手順

### Step 1: VRM モデルを用意する

VRM ファイルを入手します。無料で使えるモデルは以下から入手できます：

- **VRoid Hub**: https://hub.vroid.com/  
  → ダウンロードボタンから `.vrm` ファイルを取得

- **Booth**: https://booth.pm/ で「VRM 無料」検索

### Step 2: VRM モデルを読み込む

1. `[VRM 読み込み]` ボタンをクリック
2. ファイル選択ダイアログで `.vrm` ファイルを選択
3. ステータスが「準備完了」になったら OK

### Step 3: カメラを起動してトラッキング開始

1. `[カメラ開始]` ボタンをクリック
2. ブラウザのカメラ許可ダイアログで「許可」を選択
3. 初回は MediaPipe のモデルをダウンロードするため数十秒かかります
4. ステータスが「トラッキング中」になったら完了

### Step 4: 動かしてみる

カメラの前に顔と上半身が映るように立つと、3D モデルが動きに追従します。

| 動き | 連動する箇所 |
|------|------------|
| 顔の向き（上下・左右・傾き） | 頭・首ボーン |
| 瞬き | blinkLeft / blinkRight 表情 |
| 口の開き・形 | aa / ih / ou / ee / oh 表情 |
| 腕の上げ下げ | 上腕・前腕ボーン |
| 体の傾き | 腰・脊椎ボーン |

### Step 5: スクリーンショットを撮る

`[スクリーンショット]` ボタンをクリックすると 3D モデル画面を PNG でダウンロードします。

## カスタマイズ例

### カメラ解像度の変更

[src/camera.ts](../src/camera.ts) の `getUserMedia` オプションを変更：

```typescript
video: { width: 1280, height: 720, facingMode: 'user' }
```

### 追従の滑らかさ調整

[src/rigger.ts](../src/rigger.ts) の `LERP` 定数を変更（0〜1、小さいほど滑らか）：

```typescript
const LERP = 0.3  // デフォルト
const LERP = 0.1  // より滑らか（遅延が増える）
const LERP = 0.7  // より素早い（カクつきやすい）
```

### 背景色の変更

[src/renderer.ts](../src/renderer.ts) で `scene.background` を設定：

```typescript
// 緑背景（クロマキー合成用）
scene.background = new THREE.Color(0x00ff00)

// 透明（デフォルト）
// renderer に alpha: true が設定済み
```
